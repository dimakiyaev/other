-- Установка темы oh-my-zsh в babun. Проверено на Win10
установить babun

внести изменение в .zshrc:
ZSH_THEME="agnoster"

Скачать и установить шрифты 
git clone https://github.com/powerline/fonts
cd fonts
./install.sh

Также установить шрифты в Windows

Внести изменения в файл ~/.minttyrc
BoldAsFont=no
Columns=150
Rows=55
Font=DejaVu Sans Mono for Powerline
FontHeight=10
Transparency=low
